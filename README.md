# Portfolio Vue.js

## Description
Portfolio personnel développé avec Vue.js permettant de présenter mon profil, mes compétences et mes projets.

## Technologies utilisées
- Vue.js
- HTML
- CSS
- JavaScript

## Prérequis
- Node.js
- npm

## Installation
1. Cloner le repository :
   git clone https://github.com/userXXIIXII/my-portfolio-vue
2. Installer les dépendances :
   npm install
3. Lancer le projet :
   npm run dev
